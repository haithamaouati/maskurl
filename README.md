## Installation

1. Update and upgrade packages
    ```
    pkg update -yy
    ```
    ```
    pkg upgrade -yy
    ```
    
2. Install git package
    ```
    pkg install git -yy
    ```

3. Clone the repository
    ```
    git clone https://github.com/haithamaouati/maskurl.git
    ```
4. Change to maskurl directory
    ```
    cd maskurl
    ```
    
5. Change the access mode
    ```
    chmod +x maskurl.sh
    ```

## Usage

6. Run the script

    ```
    bash maskurl.sh
    ```
